\section{Современные архитектуры нейросетевых компрессоров на базе VAE}

\subsection{Модель с гиперприоритетом (Scale Hyperprior)}
Архитектура с гиперприоритетом, представленная в работе Ballé et al. (2018), стала ключевым прорывом в области нейросетевого сжатия. Она решает фундаментальную проблему базовой VEA — предположение о независимости элементов латентного представления $\mathbf{\hat{y}}$, что не соответствует реальной статистике изображений.

Модель вводит иерархическое вероятностное представление:

\begin{itemize}
\item Основной латентный поток $\mathbf{\hat{y}}$: Кодирует структурную информацию об изображении.
\item Гиперлатентный поток $\mathbf{\hat{z}}$: Моделирует пространственные зависимости в $\mathbf{\hat{y}}$ путем предсказания его параметров (масштаба $\sigma$ и, опционально, среднего $\mu$).
\end{itemize}

Формально, распределение основного латента задаётся как условное:
\begin{equation}
p_{\mathbf{\hat{y}}|\mathbf{\hat{z}}}(\mathbf{\hat{y}}|\mathbf{\hat{z}}) = \prod_i \mathcal{N}\big(\mu_i(\mathbf{\hat{z}}),, \sigma_i^2(\mathbf{\hat{z}})\big),
\end{equation}
где параметры $\mu_i$ и $\sigma_i$ предсказываются небольшим гипердекодером на основе сжатого гиперлатента $\mathbf{\hat{z}}$. Это позволяет более точно оценивать энтропию каждого элемента $\hat{y}_i$, учитывая его контекст, что ведёт к значительному снижению битрейта при том же уровне искажений. Данная архитектура стала де-факто стандартом для последующих исследований.

\subsection{Гибридные модели с контекстным авторегрессивным моделированием}
Дальнейшее улучшение энтропийного моделирования было достигнуто за счёт внедрения механизмов контекстного предсказания, предложенных Minnen et al. (2018). Если гиперприоритет моделирует глобальную статистику, то контекстная модель улавливает локальные пространственные зависимости.

\begin{itemize}
\item \textbf{Принцип работы:} Вероятность текущего латентного элемента $\hat{y}i$ предсказывается не только на основе гиперлатента $\mathbf{\hat{z}}$, но и с учётом уже декодированных (соседних) элементов $\hat{y}{<i}$. Это реализуется с помощью маскированных свёрточных слоёв, которые обеспечивают причинность (causality).
\item \textbf{Архитектурная реализация:} В гибридной модели энтропийный параметр $\sigma_i$ (а иногда и $\mu_i$) вычисляется как:
\begin{equation}
\sigma_i, \mu_i = f_{\text{hyper}}(\mathbf{\hat{z}}) + f_{\text{context}}(\mathbf{\hat{y}}{<i}),
\end{equation}
где $f{\text{context}}$ — авторегрессивная сеть. Это позволяет достичь наилучших на сегодняшний день показателей сжатия.
\item \textbf{Компромисс:} Несмотря на эффективность, авторегрессивный компонент делает декодирование последовательным, что замедляет его. Для ускорения используются трюки, такие как параллельное декодирование групп элементов (channel-wise context).
\end{itemize}

\subsection{Генеративно-состязательный подход (GAN) для перцептивного качества}
Для задач, где приоритетом является визуальное, а не математическое качество, применяются методы, основанные на генеративно-состязательных сетях. Архитектуры типа HiFiC (High-Fidelity Generative Image Compression) используют VAE-компрессор в качестве кодера, а декодер заменяют на генератор GAN.

\begin{itemize}
\item \textbf{Принцип:} Дискриминатор GAN обучается отличать исходные изображения от восстановленных, побуждая генератор (декодер) создавать визуально правдоподобные детали даже при высоких уровнях сжатия.
\item \textbf{Функция потерь:} Общая функция потерь комбинирует rate-loss, искажение (MSE или LPIPS) и состязательную loss:
\begin{equation}
\mathcal{L}{\text{total}} = \lambda_R R + \lambda_D D + \lambda_G \mathcal{L}{\text{GAN}}.
\end{equation}
\item \textbf{Результат:} Визуально такие методы «выигрывают» у традиционных, особенно на очень низких битрейтах, генерируя реалистичные текстуры, хотя объективные метрики типа PSNR могут быть ниже.
\end{itemize}

\subsection{Трансформеры в нейросетевом сжатии}
С появлением архитектуры Transformer начались эксперименты по её применению для сжатия изображений. Vision Transformers (ViT) используются в качестве энкодера/декодера для захвата глобальных зависимостей, а также в роли энтропийных моделей.

\begin{itemize}
\item \textbf{Преимущества:} Глобальное поле восприятия (global receptive field) с самого начала обучения против локального у свёрточных сетей. Потенциально лучшее моделирование сложных структур.
\item \textbf{Вызовы:} Высокая вычислительная сложность для изображений высокого разрешения, необходимость в больших объёмах данных для обучения. Требуется применение специальных техник, таких как оконное внимание (Swin Transformer) или патчирование.
\end{itemize}

Таким образом, современные архитектуры нейросетевых компрессоров на базе VAE представляют собой сложные иерархические системы, комбинирующие преимущества различных подходов. Эволюция идёт по пути добавления условных вероятностных моделей (гиперприоритет, контекст) для уточнения энтропийного кодирования и использования новых парадигм (GAN, трансформеры) для целевой оптимизации под конкретные задачи — будь то максимальная математическая эффективность или наилучшее перцептивное качество. Модель с гиперприоритетом и контекстным авторегрессивным компонентом остаётся эталоном для задач сжатия общего назначения.
